## üîß Troubleshooting

**Tables show "No data"**:
- Click **Sync** button on table header
- Sample data should load automatically
- If still empty, check pack is properly installed

**Can't find a table**:
- Go to **Insert** ‚Üí **Table** ‚Üí **CSM Intelligence Platform**
- Scroll through list of 16 tables
- Select table you need

**Filters not working**:
- Check column names match exactly (case-sensitive)
- Verify field types (dates as dates, numbers as numbers)
- Use table options menu (‚ãÆ) to configure

**Want to start over**:
- Delete all tables from doc
- Re-insert tables you need
- Sample data will reload fresh

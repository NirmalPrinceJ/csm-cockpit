# MuleSoft CS Platform Obsidian Template

## What This Package Includes
- `templates/MuleSoft CS Platform.md`: ready-to-use project template optimized for the MuleSoft Customer Success Intelligence Platform implementation.

## Import Instructions
1. Download or copy this folder into your Obsidian vault.
2. Place the `templates/MuleSoft CS Platform.md` file inside your vault's Templates folder (or update Obsidian's `Templates` plugin to point at `/templates`).
3. Open Obsidian and run the **Insert Template** command, selecting "MuleSoft CS Platform".
4. Duplicate the generated note for each customer or implementation phase as needed.

Optional: Keep this package in a dedicated vault (`MuleSoft CS Platform`) so you can clone or share the entire setup quickly.
